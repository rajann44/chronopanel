# Store Assets

This folder contains lightweight HTML/CSS templates for Chrome Web Store promo art.

## Files

- `promo.html`
- `promo.css`

## How to Use

1. Open `promo.html` in a browser.
2. Resize the browser window so the target asset fits cleanly.
3. Capture the large or small panel area.
4. Export or crop to the Chrome Web Store size you need.

## Suggested Targets

- Small promo tile: 440 x 280
- Large promo tile: 920 x 680
- Marquee promo tile: 1400 x 560

## Notes

- These templates are intentionally simple and static.
- Replace copy if needed to match your final store listing language.
- Keep the final asset visually consistent with the extension screenshots and website.
