What It Does

Designed for a quick, readable check-in.

Daily summary

A clean overview of total time, top sites, categories, and time-of-day patterns.

7-day trends

Move across the week and let the dashboard follow the day you choose.

Local-first privacy

History is processed inside the extension and stored in Chrome storage on your device.

Privacy

Nothing leaves your browser unless you export it yourself.

Collected: visit timestamps, domains, optional full URLs if you enable them.
Not collected: passwords, form data, message content, page content.
Sent externally: nothing.
Stored: locally in Chrome storage.

Install

Ready to publish and easy to deploy.

  1. Load the unpacked extension in Chrome during development.
  2. Run the packaged zip for Chrome Web Store submission.
  3. Deploy this static site on Vercel by setting the root directory to `website`.
