## Short Description

Private browser time dashboard with daily summaries built from Chrome history.

## Full Description

ChronoPanel helps you understand your browsing habits with a quiet, readable dashboard that stays local to your device.

It syncs recent Chrome history into daily summaries so you can review:

- total browsing time
- 7-day activity trends
- top websites
- category breakdowns
- time-of-day patterns

Why people use it:

- it stays local and simple
- it avoids cloud accounts and analytics systems
- it gives a fast daily snapshot instead of noisy reports
- it is open-source and auditable: https://github.com/rajann44/chronopanel

Privacy:

- no external server
- no account system
- no page content collection
- no passwords or form data
- optional full URL tracking is disabled by default

Important note:

ChronoPanel estimates usage from Chrome history timestamps. It is designed for lightweight habit review, not exact foreground screen-time measurement.
