# Store Assets Guide

## Chrome Web Store Sizes to Prepare

### Required

- Extension icon: 128 x 128
- At least 1 screenshot

### Recommended

- Screenshots: 1280 x 800 or 640 x 400
- Small promo tile: 440 x 280
- Large promo tile: 920 x 680
- Marquee promo tile: 1400 x 560

## Screenshot Set

Capture these screens from the extension UI with realistic data loaded.

1. Summary screen
- Show the hero ring, 7-day activity, and category bars
- Prefer a day with multiple categories and visible top sites

2. Selected-day interaction
- Click a non-today bar in the 7-day chart
- Capture the updated summary for that selected day

3. Website breakdown
- Show ranked domains, percentages, and session counts
- Use a search term only if it still looks realistic

4. Settings and privacy
- Show history sync controls, category rules, and privacy explanation text

## Screenshot Style Notes

- Use Chrome on macOS for the cleanest toolbar and side panel framing
- Keep browser zoom at 100%
- Avoid cluttered bookmarks bar content
- Prefer soft sample data with 4-8 visible domains
- Avoid empty states in store screenshots
- Use light mode for the store listing unless your dark mode looks clearly better

## Copy Angles

### Headline Ideas

- Private browser time, clearly summarized
- A calmer view of your browsing habits
- Local browser history, turned into simple daily insight

### Supporting Lines

- Built from Chrome history, stored locally on-device
- Daily totals, weekly trends, top websites, and category breakdowns
- No account, no cloud sync, no external analytics

## Promo Graphic Direction

Use the files in `store-assets/` to generate simple promo graphics that match ChronoPanel.

- `promo.html` contains ready-made large and small promo layouts
- Open it in a browser and capture the layout area you want
- Export at the target dimensions above

## Final QA Before Upload

- Confirm screenshots match the current product UI
- Confirm the privacy policy URL is live
- Confirm promo art text matches the extension description
- Confirm icon colors match the extension icon and website branding
